# User Manual - CyberGuard System

## Getting Started

### 1. Account Registration
1. Visit the homepage
2. Click "Get Started" or "Register"
3. Fill in your details:
   - Full Name
   - Email Address
   - Phone Number (optional)
   - Password (minimum 6 characters)
4. Click "Create Account"
5. Login with your credentials

### 2. Reporting a Cybercrime

#### Step 1: Access Report Form
- Click "Report Crime" from homepage or dashboard
- Login required for report submission

#### Step 2: Fill Report Details
- **Title:** Brief description of the incident
- **Incident Type:** Select from dropdown:
  - Phishing
  - Identity Theft
  - Financial Fraud
  - Cyberbullying
  - Ransomware
  - Social Media Fraud
  - Online Shopping Fraud  
  - Email Scam
  - Other

#### Step 3: Provide Description
- Detailed explanation of what happened
- Include dates, times, and relevant details
- Be as specific as possible

#### Step 4: Upload Evidence (Optional)
- Supported formats: JPG, PNG, GIF, PDF
- Maximum file size: 5MB
- Screenshots, emails, or documents

#### Step 5: Submit Report
- Review all information
- Click "Submit Report"
- Note your report ID for tracking

### 3. Tracking Your Reports

#### View All Reports
- Go to "My Reports" from dashboard
- See list of all submitted reports
- Check current status of each report

#### Report Status Meanings
- **Pending:** Report received, awaiting review
- **Under Review:** Being investigated by authorities
- **Resolved:** Case closed with resolution
- **Rejected:** Report did not meet criteria

#### View Report Details
- Click on any report to see full details
- View admin responses and updates
- Download your submitted evidence

### 4. Prevention Tips
- Access "Tips" section for cybersecurity advice
- Learn about common scams and how to avoid them
- Stay updated with latest security practices

## Admin Functions (Admin Users Only)

### Dashboard Overview
- View total reports statistics
- Monitor pending cases
- Quick access to recent reports

### Managing Reports
1. **View All Reports**
   - Access complete list of user reports
   - Filter by status or date
   - Search by keywords

2. **Update Report Status**
   - Open report details
   - Change status (Pending → Review → Resolved/Rejected)
   - Add admin response/notes

3. **Respond to Users**
   - Provide updates on investigation progress
   - Request additional information if needed
   - Notify users of case resolution

### Managing Tips
- Add new prevention tips
- Edit existing content
- Organize tips by category
- Keep content current and relevant

## Troubleshooting

### Common Issues

**Cannot Login**
- Verify email and password
- Check for typos
- Use "Forgot Password" if available

**File Upload Fails**
- Check file size (max 5MB)
- Verify file format (JPG, PNG, GIF, PDF only)
- Try a different file

**Report Not Visible**
- Ensure you're logged into correct account
- Check "My Reports" section
- Contact admin if report is missing

### Contact Support
- Email: support@cyberguard.com
- Emergency Cybercrime Helpline: 1930
- Report issues through the system

## Security Best Practices
1. Use strong, unique passwords
2. Never share your login credentials
3. Log out when using public computers
4. Keep evidence files secure
5. Report suspicious activities immediately
